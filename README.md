# 3D Portfolio

Interactive developer portfolio built with Three.js and React. Features a real-time 3D scene that showcases tech stack and projects.

## What's inside

- Three.js 3D scene with orbital controls
- Animated tech-stack icons floating in 3D space
- Project cards with live links
- Responsive — desktop and mobile
